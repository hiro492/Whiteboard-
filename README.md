# お絵かき斜面ボールゲーム

ホワイトボードに描いた線が、そのまま物理演算の「地形」になるブラウザゲームです。マウスで線を描いてボールを転がせるほか、黒ペンで描いた実物のホワイトボードをカメラやアップロード画像から検出して地形化し、プロジェクターで投影しながら遊ぶことを想定しています。

## 特徴

- **マウス描画**：ドラッグで自由に線を描き、その上をボールが転がる
- **画像から地形検出**：アップロードした画像やカメラ映像を OpenCV.js で解析し、黒ペンの線を検出して地形に変換
- **リアルタイムモード**：一定間隔でカメラ映像を再キャプチャし、地形を継続的に更新
- **ROI(検出範囲)指定**：カメラプレビュー上をドラッグして処理対象範囲を絞り込み
- **フルスクリーン投影**：物理演算の内部解像度(800×600)はそのままに、CSSで見た目だけ拡大してプロジェクターへ投影
- **プロカム調整(キーストーン補正)**：斜め投影による台形歪みをホモグラフィー変換で補正
  - 四隅ドラッグによる手動調整
  - カメラ映像を使った自動調整(白黒パターンを投影して自動検出)
- **出力モニター選択**：複数ディスプレイ環境で投影先のモニターを指定してフルスクリーン表示

## 動作環境

- ビルド不要・パッケージマネージャ不要。静的ファイル(HTML/JS)を配信するだけの構成です
- p5.js / matter.js / OpenCV.js を CDN から読み込みます(インターネット接続が必要)
- カメラ機能(`getUserMedia`)や CDN 読み込みのため、**HTTPサーバー経由での配信が必須**です(`file://` では動作しません)
- 出力モニター選択機能(Window Management API)は **Chrome / Edge 等 Chromium系ブラウザ**のみ対応。非対応ブラウザでは既定モニターへの通常フルスクリーンにフォールバックします

## 起動方法

`whiteboard-game/` フォルダを任意の静的サーバーで配信してください。例えば:

```bash
npx http-server whiteboard-game -p 8123 -c-1
```

起動後、ブラウザで `http://localhost:8123` を開きます。

## 使い方

### 1. マウスで描く
キャンバス上をドラッグすると線を描けます。スペースキーでマウス位置からボールを落とせます。

### 2. 画像から地形を作る
「入力ソース」で「ファイル」を選び、「画像を選択」から黒ペンで描いた画像をアップロードします。各種スライダー(検出方式、彩度/明度しきい値、輪郭の長さ・面積、モルフォロジー処理など)で検出結果を調整できます。

### 3. カメラから地形を作る
「入力ソース」で「カメラ」を選び、「カメラ開始」を押します。プレビュー上をドラッグすると検出範囲(ROI)を指定できます。「この映像で地形を作る」で一度だけ検出、「リアルタイム開始」で一定間隔ごとに自動再検出します。

### 4. フルスクリーン投影
「フルスクリーン」ボタンでゲーム画面のみを全画面表示します。プロジェクターに投影する際に使用します。

### 5. プロカム調整(投影の歪み補正)
「プロカム調整」ボタンで調整モードに入ります。

- **手動調整**：表示される4つのハンドルを、実際に投影された映像を見ながら投影面の角に合わせてドラッグします
- **カメラで自動調整**：正面から投影面を写すカメラを起動しておき、「カメラで自動調整」を押すと白黒パターンを一瞬投影してカメラ画像から自動的に補正します。自動調整後、必要であれば手動でさらに微調整できます
- 調整後は「保存」を押すと補正内容が保存され(単一スロット)、次回のフルスクリーン投影時にも自動的に適用されます

### 6. 出力モニターの選択
複数モニター環境では、「投影出力」欄の「モニター一覧を更新」でモニターを検出し、プルダウンで選択後「選択モニターに投影」を押すと、指定したモニターへフルスクリーン投影します。

## 技術構成

- **p5.js**：描画ループとキャンバス管理
- **matter.js**：物理演算(線は静的ボディ、ボールは動的ボディ)
- **OpenCV.js**：画像/カメラ映像からの線検出(黒ペンHSVマスク、または適応的二値化)
- ビルドツールなし、フレームワークなしの素の HTML + JavaScript
