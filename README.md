# お絵かき斜面ボールゲーム

ホワイトボードに描いた線が、そのまま壁となり自作のピタゴラスイッチを作ることのできる作品です。

黒いペンでホワイトボードに線を描くと、カメラがその線を読み取って、画面の中にそっくりなを作ります。あとはボールを落とすだけで、描いた線の上をコロコロ転がっていきます。プロジェクターでホワイトボードに映像を映しながら遊ぶことを想定しています。

マウスで直接キャンバスに線を描くこともできます。

## できること(かんたんまとめ)

- **マウスで線を描く**：画面をドラッグすると線が引けて、ボールがその上を転がる
- **画像から坂道を作る**：写真をアップロードすると、写っている黒い線を自動で坂道に変換する
- **カメラで坂道を作る**：パソコンのカメラでホワイトボードを映すと、リアルタイムで坂道が作られる
- **カメラの傾き補正**：カメラを斜めに置いても、映像を四角くまっすぐに直してから読み取れる
- **フルスクリーン表示**：ゲーム画面だけを画面いっぱいに表示できる(プロジェクター投影用)
- **プロジェクターの歪み補正**:プロジェクターを斜めから投影しても、映像の形をまっすぐに直せる
- **2画面出力**:パソコンの画面には操作パネルを表示したまま、プロジェクターにはゲーム画面だけを映せる
- **効果音**:ボールが線にぶつかると「コツン」、線の上を転がると「ゴロゴロ」という音が鳴る

## 動かすのに必要なもの

- 特別な準備は不要です。インストール作業もありません(HTML と JavaScript のファイルをそのまま使います)
- ただしインターネットには繋がっている必要があります(必要な部品をネットから読み込むため)
- カメラを使う機能や、ネットから部品を読み込む都合上、**このゲームは「サーバー」を通して開く必要があります**。パソコンの中のファイルをブラウザでそのまま開く(ダブルクリックする)だけでは正しく動きません
- 出力モニターを選ぶ機能は、Google Chrome や Microsoft Edge など一部のブラウザでのみ使えます。それ以外のブラウザでは、この機能は使えませんが、それ以外は問題なく遊べます

## 起動のしかた

### かんたんな方法(Windows・おすすめ)

`ゲームを起動.bat` を**ダブルクリック**するだけです。サーバーが起動し、自動で既定のブラウザに `http://localhost:8081` が開いてゲームが始まります。閉じるときは、開いた黒いウィンドウを閉じてください。

- **Node.js のインストールは不要です。** Windows に最初から入っている PowerShell だけで動きます
- どのフォルダに置いても、ショートカットから起動しても動きます(場所に依存しません)
- うまくいかないときは、`ゲームを起動.bat` を右クリック →「管理者として実行」を試してください

### コマンドで起動する方法(Node.js がある場合)

パソコンにサーバーを立てて、`whiteboard-game` フォルダを配信します。ターミナル(コマンドを打つ画面)で次のコマンドを実行してください。
Windowsの場合は、PowerShellを使ってこのコマンドをプログラムのあるフォルダで実行します。

```bash
npx http-server whiteboard-game -p 8123 -c-1
```

実行すると、ブラウザで `http://localhost:8123` を開けばゲームが始まります。

## 使い方

### 1. マウスで線を描いて遊ぶ
画面をドラッグすると線が引けます。スペースキーを押すと、マウスがある場所からボールが落ちてきます。

### 2. 写真から坂道を作る
「入力ソース」で「ファイル」を選び、「画像を選択」ボタンから、黒いペンで線を描いた写真を選びます。うまく検出できないときは、下にあるスライダー(つまみ)を動かして調整できます。

### 3. カメラで壁を作る
「入力ソース」で「カメラ」を選び、「カメラ開始」ボタンを押します。パソコンのカメラがホワイトボードを映します。

- 「この映像で地形を作る」ボタン：今映っている映像から、1回だけ坂道を作ります
- 「リアルタイム開始」ボタン：一定の間隔で自動的に映像を読み取り続け、壁を更新し続けます

### 4. カメラを斜めに置いた場合の補正
カメラを正面からではなく斜めに置くと、四角いホワイトボードが台形にゆがんで映ってしまいます。これを直すための機能です。

1. 「カメラ4隅を合わせる」ボタンを押します
2. カメラの映像の上に、4つの丸いハンドル(つまみ)が出てきます
3. その4つのハンドルを、ホワイトボードの実際の4つの角(左上・右上・右下・左下)に合わせてドラッグします
4. もう一度「カメラ4隅を合わせる」ボタン(「カメラ4隅を確定」に変わっています)を押すと確定します

これで、斜めから撮っても映像がまっすぐな四角形に直ってから読み取られるようになります。設定はブラウザに保存されるので、次に開いたときも引き継がれます。

「カメラ補正リセット」ボタンを押すと、この補正を解除して元の映像全体を使う状態に戻せます。

### 5. フルスクリーン表示
「フルスクリーン」ボタンを押すと、ゲーム画面だけが画面いっぱいに表示されます。プロジェクターに映すときに使います。

### 6. プロジェクターの歪み補正
プロジェクターを斜めから投影すると、映像が台形にゆがんでしまいます。これを直すための機能です。

1. 「プロジェクタ調整」ボタンを押します
2. 画面の上に、4つの丸いハンドルが出てきます
3. 実際にプロジェクターで投影されている映像を見ながら、その4つのハンドルを、投影したい面(ホワイトボードなど)の実際の4つの角に合わせてドラッグします
4. 「保存」ボタンを押すと、この設定が保存されます。次回フルスクリーン表示したときも、自動でこの補正が使われます
5. 「終了」ボタンで調整モードを抜けます

「カメラを斜めに置いたときの補正」と「プロジェクターを斜めに置いたときの補正」は、まったく別々の機能です。それぞれ別に調整してください。

### 7. 効果音のON/OFFと音量調整
ボールが線にぶつかったときの「コツン」という音と、線の上を転がっているときの「ゴロゴロ」という音が鳴ります。

- 画面上の「効果音」チェックボックスを外すと、音を消せます
- 隣の音量スライダーで、音の大きさを調整できます
- 設定はブラウザに保存されるので、次に開いたときも引き継がれます
- 音はブラウザの仕様上、画面を一度クリックするかキーを押すまでは鳴りません(最初にドラッグで線を描くか、スペースキーでボールを落とせば、それ以降は鳴るようになります)

### 8. 2つの画面に別々のものを映す(2画面出力)
ノートパソコンの画面には操作パネルを表示したまま、プロジェクターにはゲーム画面だけを映したいときに使います。

1. 複数のモニターを繋いでいる場合、「モニター一覧を更新」ボタンを押すと、繋がっているモニターの一覧が出ます
2. プルダウンからプロジェクターを選びます(選ばなければ、いつも使っているモニターに出ます)
3. 「選択モニターに投影」ボタンを押すと、選んだモニターに新しいウィンドウが開き、ゲーム画面だけが映ります
4. そのウィンドウの中をクリックすると、そのモニターいっぱいに広がります(全画面表示)
5. もう一度ボタン(「投影を終了」に変わっています)を押すと、投影が終わります

ノートパソコン側の画面はそのまま操作パネルとして使い続けられます。

## どんな仕組みで動いているか(かんたんに)

- **p5.js**という部品が、画面の絵を描いたり、ボールを動かしたりしています
- **matter.js**という部品が、ボールが転がる・線にぶつかるといった物理的な動きを計算しています
- **OpenCV.js**という部品が、写真やカメラの映像から「黒い線」を見つけ出しています
- ボールがぶつかる音・転がる音は、音声ファイルを使わず**ブラウザの音声機能(WebAudio)**でその場で作っています
- 特別なソフトのインストールや、複雑な準備は必要ありません。HTML と JavaScript だけでできています

もっと詳しい仕組みや、開発者向けの注意点は [CLAUDE.md](CLAUDE.md) に書いてあります。
